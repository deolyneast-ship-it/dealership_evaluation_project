# Car Dealership Central
**Project Name:** Cars Dealership Full-Stack Management Portal

## 🚗 Project Overview
This repository contains the Full-Stack Capstone Project for the IBM Full Stack Software Developer Professional Certificate. The application is a comprehensive platform for a national car retailer, allowing users to browse dealership locations, view car models, and submit/view reviews with integrated sentiment analysis.

## 🛠️ Tech Stack
- **Frontend:** React.js, HTML5, CSS3, Bootstrap
- **Backend:** Django (Web Server), Node.js (Dealership/Review Service), Flask (Sentiment Analysis)
- **Databases:** MongoDB (Reviews/Dealers), SQLite (Car Models/User Data)
- **Deployment:** Docker, Kubernetes, IBM Cloud Code Engine
- **CI/CD:** GitHub Actions

## 📂 Repository Structure
- `/server`: Django application and static files.
- `/server/frontend`: React application source code.
- `/server/djangoapp`: Backend logic and API routes.
- `/functions`: Serverless functions for sentiment analysis and database interactions.

## 🚀 Key Features
- Responsive Navigation Bar and "About/Contact" pages.
- User Authentication (Sign-up, Login, Logout).
- Dynamic Dealer and Review management.
- Sentiment analysis on user reviews (Positive, Neutral, Negative).
- Fully containerized deployment using Docker.
